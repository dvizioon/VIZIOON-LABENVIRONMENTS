================================================================================
PUBLICAR A PÁGINA DE PRIVACIDADE NO GITHUB
================================================================================

PASSO 1 — ATIVAR PAGES (OBRIGATÓRIO, faz só uma vez)
----------------------------------------------------

Abra:
https://github.com/dvizioon/VIZIOON-LABENVIRONMENTS/settings/pages

Em "Build and deployment" → Source / Origem:

  Selecione: GitHub Actions

(NÃO use "Deploy from a branch" na primeira vez.)

Salve. Se não aparecer "GitHub Actions", o repositório precisa ser público
ou ter plano que inclua Pages em repo privado.


PASSO 2 — RODAR O WORKFLOW
--------------------------

Opção A: push na branch main (dispara automático)
Opção B: Actions → "Deploy GitHub Pages" → Run workflow


PASSO 3 — URL FINAL
-------------------

https://dvizioon.github.io/VIZIOON-LABENVIRONMENTS/privacidade.html

Cole na Chrome Web Store (Política de Privacidade).


================================================================================
ERRO 404 NO deploy-pages? (Creating Pages deployment failed)
================================================================================

Causa: GitHub Pages NÃO está ativado ou a origem não é "GitHub Actions".

Correção:
1. Settings → Pages → Source = GitHub Actions
2. Actions → Deploy GitHub Pages → Re-run all jobs

Se ainda falhar:
- Confirme que o repo é público (ou Pages habilitado no plano)
- Confirme que a branch é main (o workflow escuta main e master)
- Espere 1–2 min após ativar Pages e rode de novo
