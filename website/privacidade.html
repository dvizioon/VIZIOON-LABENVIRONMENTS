Política de Privacidade

Última atualização: junho de 2025

Resumo: a extensão não vende dados, não envia variáveis .env para servidores do desenvolvedor e guarda configurações apenas no seu navegador.

1. Introdução

A extensão VIZIOON Lab Environments (“Extensão”) é desenvolvida por Daniel Martins (contato: danielmartinsjob@gmail.com). Esta política descreve como a Extensão lida com informações quando você a utiliza no Google Chrome.

2. O que a Extensão faz

A Extensão ajuda desenvolvedores a importar variáveis de ambiente de arquivos .env para a página CI/CD → Variables do GitLab, diretamente no navegador. Toda a operação principal ocorre na página GitLab que você abrir voluntariamente.

3. Dados que a Extensão acessa

3.1 Armazenamento local

Usamos chrome.storage.local apenas no seu dispositivo para:

  • Preferência do botão flutuante (ligado/desligado)
  • Opção “salvar automaticamente”
  • Abas de ambiente criadas por você
  • Fila de variáveis importadas do .env (por ambiente)

3.2 Conteúdo da página GitLab

Quando você usa a Extensão na página Variables do GitLab, ela pode ler:

  • Nomes (chaves) de variáveis já cadastradas na página
  • Se a variável está marcada como Protected

Essa leitura ocorre somente no seu navegador para exibir a lista e para preencher ou remover linhas conforme sua ação.

3.3 Arquivos .env

Arquivos .env são lidos apenas quando você os seleciona ou arrasta para a Extensão. O conteúdo é processado localmente para montar a fila de importação.

4. Dados que não coletamos

Não coletamos intencionalmente:

  • Nome, e-mail ou outros dados de identificação pessoal
  • Senhas ou tokens do GitLab (a Extensão não solicita token de API)
  • Histórico de navegação
  • Localização ou endereço IP para rastreamento
  • Dados de saúde, financeiros ou comunicações pessoais

5. Transmissão a terceiros

A Extensão não transmite variáveis de ambiente, filas .env ou conteúdo da página GitLab para servidores do desenvolvedor.

Única comunicação de rede opcional:

  • API pública do GitHub (api.github.com) na aba Versões, para listar releases públicas do repositório. Nenhum dado pessoal do usuário é enviado nessa requisição.

6. Permissões do Chrome

  • storage — salvar configurações e filas localmente
  • tabs — abrir o painel na aba ativa e exibir badge ON/OFF
  • host (*://*/*) — injetar a interface na página GitLab Variables (inclui GitLab self-hosted com domínios variados)

7. Retenção e exclusão

Dados locais permanecem no dispositivo até você:

  • Limpar filas e configurações pela própria Extensão
  • Remover dados da extensão em chrome://extensions
  • Desinstalar a Extensão

8. Menores de idade

A Extensão é uma ferramenta profissional para desenvolvedores e não é direcionada a menores de 13 anos.

9. Alterações

Podemos atualizar esta política. A versão vigente estará nesta página e no repositório GitHub do projeto.
