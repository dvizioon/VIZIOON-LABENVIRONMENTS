# Site — VIZIOON LAB ENVIRONMENTS

Documentação em [Docusaurus](https://docusaurus.io/).

```bash
npm install
npm start      # dev em localhost:3000
npm run start:clean  # se der erro de cache no Windows
npm run build  # gera website/build
```

Screenshots: coloque PNGs em `static/img/screenshots/` (lista em `static/img/screenshots/README.md`).

Deploy: push na `main` → workflow `.github/workflows/pages.yml` publica em GitHub Pages.
