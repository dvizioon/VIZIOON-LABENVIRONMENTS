# Navegadores compatíveis

A extensão roda em navegadores Chromium com suporte a extensões Manifest V3.

<BrowserCompat />

Use uma versão recente do navegador. A extensão precisa abrir páginas do GitLab normalmente, como qualquer aba.

Para instalar, veja [Instalação](/docs/instalacao).
