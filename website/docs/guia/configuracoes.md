# Configurações

Na aba **Config** você ajusta como a extensão se comporta.

<Screenshot
  file="aba-config.png"
  alt="Barra lateral — aba Config"
/>

## Auto-save

Com essa opção ligada, a extensão salva cada variável no GitLab automaticamente ao preencher a página. Desligada, você revisa e confirma cada uma no próprio GitLab.

## Botão flutuante

No popup da extensão, na barra do navegador, há um interruptor para mostrar ou esconder o botão flutuante nas páginas do GitLab.

<Screenshot
  file="popup-toggle-flutuante.png"
  alt="Popup — interruptor do botão flutuante"
/>

## Dados salvos

Abas, configurações e histórico ficam no navegador. Se limpar os dados da extensão, tudo volta ao padrão.
