# Histórico

A aba **Histórico** guarda as importações que você já fez.

<Screenshot
  file="aba-historico.png"
  alt="Barra lateral — aba Histórico"
/>

## Quando aparece um registro

Sempre que você importa um `.env` ou preenche a página com variáveis na fila, a extensão salva aquela sessão com data, ambiente usado e lista de variáveis.

## Usar de novo

O botão **Usar** recoloca aquela importação na fila da aba Variáveis, no mesmo ambiente de quando foi salva.

<Screenshot
  file="historico-usar.png"
  alt="Histórico expandido com botão Usar"
/>

## Exportar e importar

Você pode baixar o histórico em JSON ou carregar um arquivo exportado antes. Serve para backup ou para levar a configuração para outro computador.

## Limpar

Dá para apagar entradas uma a uma ou limpar tudo. Isso só remove o que está salvo no seu navegador.
